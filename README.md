# Тверская площадь — Инфографика

Одностраничная инфографика по результатам опроса жителей Твери
о благоустройстве Тверской площади (148 респондентов, 19 вопросов).

## Стек

- Чистый HTML / CSS / JS — без зависимостей
- `nginx:1.27-alpine` для раздачи
- Деплой через Dokploy на `dev.ii4ki.ru`

## Локальный запуск

```bash
# просто открыть index.html в браузере
# или через любой статик-сервер
python -m http.server 8080
```

## Деплой

`Dockerfile` собирает статический nginx-образ. Докплою подключён
GitHub-провайдер — push в `main` триггерит redeploy.
